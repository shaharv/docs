Ubuntu First Steps
==================

1. Update system packages:

   $ sudo apt update && sudo apt upgrade

2. Install latest git:

   $ sudo add-apt-repository ppa:git-core/ppa
   $ sudo apt update && sudo apt-get install git

3. Configure git:

   $ git config --global user.name username
   $ git config --global user.email user@domain.com

4. ~/.bashrc convinience updates (append at EOF):

   force_color_prompt=yes
   bind TAB:menu-complete
   alias cd..="cd .."

5. Add startup scripts under /etc/profile.d - for example, setting env. variables

6. Disable sudo password for sudoers:

   - $ sudo visudo

   - Add the 3rd line below:
     # Allow members of group sudo to execute any command
     %sudo   ALL=(ALL:ALL) ALL
     shahar  ALL=NOPASSWD: ALL

7. Install basic packages:

  $ sudo apt-get install \
      audacious \
      cmake \
      coreutils \
      dos2unix \
      gitk \
      meld \
      nano \
      ninja-build \
      python3-pip \
      tree

8. Configure nano - create .nanorc:

   set tabsize 4

9. Add toolchain repository (optional):

   $ sudo add-apt-repository ppa:ubuntu-toolchain-r/test
   $ sudo apt update && sudo apt upgrade

10. Install Clang 9:

   $ sudo apt-get install clang-9 clang-tidy-9 clang-format-9
   $ sudo update-alternatives --install /usr/bin/clang clang /usr/bin/clang-9 900 --slave /usr/bin/clang++ clang++ /usr/bin/clang++-9
   $ sudo update-alternatives --install /usr/bin/clang-tidy clang-tidy /usr/bin/clang-tidy-9 900
   $ sudo update-alternatives --install /usr/bin/clang-format clang-format /usr/bin/clang-format-9 900

11. Install latest docker engine:

   https://docs.docker.com/install/linux/docker-ce/ubuntu
