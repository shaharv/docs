Firefox
-------

- Hard refresh                            Ctrl + F5


Chrome
------

Basic:

- Downloads                               Ctrl + J
- Restore recent closed tab               Ctrl + Shift + T

Developer:

- Open develper mode                      F12
- Disable page caching                    Network -> Disable cache
- Search for script files                 Ctrl + O
- Reload and refresh options              Right click refresh button

Management:

- Chrome task manager                     Shift + Esc

Javascript debugging:

- Choose file                             F12 -> Elements -> Sources
- Run                                     F8
- Step Over                               F10
- Step In                                 F11
