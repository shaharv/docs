VSCode Shortcuts
----------------

Navigate

- Go to file           Ctrl + P
- Go to line           Ctrl + G
- Navigate backwards   Alt + Left Arrow
- Navigate forward     Alt + Right Arrow
- Search all files     Ctrl + Shift + F

Run and debug

- Start debug          F5
- Stop debug           Shift + F5

Installation (Windows+WSL)
--------------------------

- Install VSCode for Windows:
  https://code.visualstudio.com/download#

- Enable VSCode WSL support:
  https://code.visualstudio.com/docs/remote/wsl

- VSCode and C++ on WSL:
  https://code.visualstudio.com/docs/cpp/config-wsl

- Install extensions (for WSL):
  - C/C++
  - C/C++ Extension Pack
  - CMake
  - CMakeTools
  - Python
  - Remote - WSL
  - TabNine
  - Code Spell Checker
