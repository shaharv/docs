VSCode Shortcuts
----------------

Navigate

- Go to file                         Ctrl + P
- Go to line                         Ctrl + G
- Move to previous location          Alt + Left Arrow
- Move to next location              Alt + Right Arrow
- Search all files                   Ctrl + Shift + F
- Replace in all files               Ctrl + Shift + H
- Switch one tab to the left         Ctrl + PageUp
- Switch one tab to the right        Ctrl + PageDown
- Switch between header/source       Alt + O

Edit

- Copy / paste the active line       Ctrl + C / Ctrl + V
- Move the active line               Alt + Up/Down Arrow
- Comment / uncomment lines          Ctrl + /
- Select next occurance of word      Ctrl + D
- Select all occurances of word      Ctrl + F2
- Rename all occurances of symbol    F2
- Mark rectangle block of text       Shift + Alt while dragging mouse
- Trigger auto-complete manually     Ctrl + Space

Edit - Multiple Cursors

- Edit multiple lines at once        Put cursor on each line with Alt + Left click, then type
- Put cursor at end of marked lines  Alt + Shift + I

Run and debug

- Start debug                        F5
- Stop debug                         Shift + F5

Commands and settings

- Open the command palette           F1
- See all keyboard shortcuts         Ctrl + K + S
- Open settings                      Ctrl + ,

View

- Hide/show left side bar            Ctrl + B
- Hide/show bottom panel             Ctrl + J

Misc

- Kill all running VSCode process    killall node
  (assuming no other node apps)

Installation (Windows+WSL)
--------------------------

- Install VSCode for Windows:
  https://code.visualstudio.com/download#

- Enable VSCode WSL support:
  https://code.visualstudio.com/docs/remote/wsl

- VSCode and C++ on WSL:
  https://code.visualstudio.com/docs/cpp/config-wsl

- Install extensions (for Windows and WSL):
  - C/C++
  - C/C++ Extension Pack
  - CMake
  - CMakeTools
  - Python
  - Remote - SSH
  - Remote - WSL
  - TabNine
  - Code Spell Checker
