vLLM info and tips
------------------

Building:

- Build and install local vLLM to system editable package      VLLM_TARGET_DEVICE=cpu python3 -m pip install -e . --no-build-isolation
  for CPU. Notes:
  1. Modify requirements/cpu* to require torch>=2.8.0
  2. Requires avx512 (check `lscpu | grep avx512`)
  3. Using no build isolation (local env) to pick right torch

- Full vLLM build instructions for CPU                         https://docs.vllm.ai/en/stable/getting_started/installation/cpu
